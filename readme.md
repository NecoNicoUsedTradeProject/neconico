# 요구사항
***
### 회원
* 회원 가입
* 로그인 및 로그아웃
* 개인정보 수정
* 판매, 구매, 찜 상품 확인
### 상품
* 상품 등록 및 수정
* 거래 요청
* 상품 찜하기
* 상품 검색
* 상품 문의 및 댓글
### 상점
* 내 상점과 다른 회원 상점 
* 상점 정보 등록 및 수정
* 거래 승인
* 상품 목록 및 정렬
### 관리자
* 회원 목록 조회
* 회원 그룹별 조회(연령/성별/지역)
* 카테고리별 등록 조회
* 거래 완료 비율 조회
* 카테고리 추가 및 수정
* 공지사항 작성, 수정, 숨기기
* 광고 작성 및 수정
